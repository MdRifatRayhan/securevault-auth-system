# 🔐 SecureVault Authentication System

A secure authentication system built with Node.js, Express.js and MongoDB.

---

## 🚀 Features

- User Registration & Login
- OTP Verification
- JWT Authentication
- CSRF Protection
- Password Hashing
- Password Reset System
- Login History Tracking
- Protected Routes
- Dark Mode UI
- MongoDB Atlas Integration

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- HTML
- CSS
- JavaScript
- JWT
- bcrypt
